# Django Template Language (DTL) Project

This repository contains a complete Django project demonstrating the use of **Django Template Language (DTL)**, including views, models, templates, static assets, and basic CRUD operations.

## Project Structure

```

DJANGO/
├── COMPLETE_DJANGO.IPYNB        # Jupyter notebook guide
├── COMPLETE_DJANGO.html         # HTML guide
├── DTL/                         # Django project folder
│   ├── DTL/                     # Project settings
│   ├── Design/                  # Django app
│   ├── Template/                # HTML templates
│   └── static/                  # Static assets (CSS, JS, images)
└── manage.py

```

## Features

- Demonstrates **Django Template Language** for dynamic content rendering.

- Contains static files (CSS, JS, images) for a fully styled UI.
- Notebook and HTML guide for step-by-step reference.

## Setup Instructions

1. **Create a virtual environment**:
   ```bash
   python -m venv env
   source env/bin/activate  # Linux/Mac
   env\Scripts\activate     # Windows
   ```

2. **Install dependencies**:

   ```bash
   pip install django
   ```

3. **Run the server**:

   ```bash
   python manage.py runserver
   ```

4. Open [http://127.0.0.1:8000](http://127.0.0.1:8000) in your browser.

## Notes

* Exclude the SQLite database and `__pycache__` from GitHub.
* Designed as a reference for learning DTL and basic Django apps.
